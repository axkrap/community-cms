TEMPLATES
=========

This file contains a list of all of the HTML comments that the template engine 
will automatically strip out and a description of the content that will replace
them. There are some tags that are globally replaced, while others are replaced
only for certain files.

GLOBAL
======
<!-- $PAGE_TITLE$ -->                Page title (usually in title bar)
<!-- $CSS_INCLUDE$ -->               Included CSS files based on the database
<!-- $IMAGE_PATH$ -->                Path to template image directory
<!-- $CONTENT$ -->                   Main page content
<!-- $FOOTER$ -->                    Page footer
<!-- $ADMIN_INCLUDE$ -->             Includes all scripts required by admins
<!-- $NOTEBOOK_CONTENT$ -->          Placeholder
<!-- $NAV_BAR$ -->                   Displays a vertical list of pages
<!-- $NAV_LOGIN$ -->                 Displays the login box, or user info

login.html
==========
<!-- $LOGIN_USERNAME$ -->            Displays a username input field.
<!-- $LOGIN_PASSWORD$ -->            Displays a password input field.
<!-- $LOGIN_BUTTON$ -->              Displays a login button.

messages.html
=============
<!-- $MESSAGE_BODY$ -->              Message body

article.html
============
<!-- $ARTICLE_DATE_MONTH$ -->        Displays the month as a number between
                                     1 and 12
<!-- $ARTICLE_DATE_MONTH_TEXT$ -->   Displays the month as a three letter
                                     abbreviation (e.g. OCT, DEC, APR)
<!-- $ARTICLE_DATE_DAY$ -->          Displays the day of the month that an
                                     article was last edited
<!-- $ARTICLE_DATE_YEAR$ -->         Year of last article edit
<!-- $ARTICLE_DATE$ -->              Article date as stored in database
<!-- $ARTICLE_IMAGE$ -->             HTML image tag for article image
<!-- $ARTICLE_AUTHOR$ -->            Name of last user to edit an article
<!-- $ARTICLE_TITLE$ -->             Displays an article's title
<!-- $ARTICLE_CONTENT$ -->           Displays content of an article
<!-- $ARTICLE_ID$ -->                Integer id of the current article

contactlist.html
================
<!-- $CONTACT_IMAGE$ -->             Reserved
<!-- $CONTACT_NAME$ -->              Name of contact in the form Last, First
<!-- $CONTACT_TITLE$ -->             Title or position of user
<!-- $CONTACT_EMAIL_START$ -->       Code between these tags will be removed
<!-- $CONTACT_EMAIL_END$ -->         if the user's email is hidden.
<!-- $CONTACT_ADDRESS_START$ -->     Code between these tags will be removed
<!-- $CONTACT_ADDRESS_END$ -->       if the user's address is hidden.
<!-- $CONTACT_TELEPHONE_START$ -->   Code between these tags will be removed
<!-- $CONTACT_TELEPHONE_END$ -->     if the user's phone number is hidden.
<!-- $CONTACT_EMAIL$ -->             Displays email address
<!-- $CONTACT_ADDRESS$ -->           Address
<!-- $CONTACT_TELEPHONE$ -->         Telephone number

pagination.html
===============
<!-- $NEXT_PAGE$ -->                 Link to next page of results
<!-- $PREV_PAGE$ -->                 Link to previous page of results
