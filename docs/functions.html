display_login_box()

 Description:
string display_login_box ( )
If you are not logged in, displays a login box. Otherwise displays user links.

Return Values:
Returns a login box based on the template file login.html, or displays a list of user links if you are logged in.

err_page()

 Description:
string err_page ( int $error_code )
Given an error code, returns an html page displaying the error message.

Parameters:
error_code
The error code, as defined in docs/errorcodes.txt. If none is specified, error code 0 will be used.
Return Values:
Returns a complete html page with an error message.

load_template_file()

 Description:
array load_template_file ( string $file_name )
Given a string with the name of the file you would like to access, returns the contents of the file, and the directory in which it was found.

Parameters:
file_name
The name of the file within the site's template directory. The correct directory will automatically be determined based on the contents of the database. If no value is specified, index.html will be used.

Return Values:
The ['content'] value of the returned array shows the content of the file. The ['template_path'] value shows the template directory.

Location:
This function is defined in functions/files.php
