$_SESSION array
Variable:Description:
$_SESSION['name']Real name of user, in last, first format, NULL if not logged in.
$_SESSION['pass']MD5 hash of user's password, NULL if not logged in.
$_SESSION['type']User authentication level. A type greater than 0 results in administrative permissions. 0 or NULL if not logged in.
$_SESSION['user']Username of currently logged in user, NULL if not logged in.
$_SESSION['userid']ID of logged in user, NULL if not logged in.

$CONFIG array
Variable:Description:
$CONFIG['db_host']Host of the MySQL database. Usually localhost.
$CONFIG['db_name']Name of the MySQL database.
$CONFIG['db_pass']Password of the MySQL database (in plain text).
$CONFIG['db_prefix']Table prefix within MySQL database. Only 'comcms_' is supported at this time.
$CONFIG['disabled']If set to 1, the site will be inaccessible.
